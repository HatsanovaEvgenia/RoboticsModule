# Формат устного зачёта

**Процедура:** студент заходит в аудиторию и вытягивает билет с **тремя теоретическими вопросами**.
  Дополнительно выдаётся **кинематическая схема** механизма.

**Практическая задача:** по выданной схеме необходимо решить **прямую и обратную задачи кинематики по положению** с использованием **метода Денавита—Хартенберга**.

**Оценивание:**

  * решение практической задачи — **4 балла**;
  * ответы на каждый из трёх вопросов билета — по **2 балла** каждый.
    **Максимальная суммарная оценка — 10 баллов.**

**Время на подготовку:** **45 минут**, после чего начинается приём ответов.
